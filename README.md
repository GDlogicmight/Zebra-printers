# Zebra-printers
Tools and scripts for managing Zebra printers 

Healthcare organizations usually have 100's if not thousands of Zebra printeres used for printing everything from patient arm-bands to specimun lables.  Managing these can sometimes require some special capabilities.  This is a collection of tools, programs and scripts to help you.

## Zebra PowerOnReset
This tool allows you to send a RESET-like command to printers on your network.
This can be helpful during troublshooting or when you need a restart to add/remove a configuation change
